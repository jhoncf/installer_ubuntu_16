# Ubuntu Installer

## Como usar

### Lamp
  
  ``` 
  ./installer lamp 
  ```

### Mongodb
  
  ``` 
  ./installer mongodb 
  ```
  
### Docker
  
  ``` 
  ./installer docker 
  ```
  
### Composer
  
  ``` 
  ./installer composer 
  ```

### Phpunit
  
  ``` 
  ./installer phpunit 
  ```

### Node.js

  ``` 
  ./installer nodejs 
  ```

### Apidoc

  ``` 
  ./installer apidoc 
  ```

### Fail2ban

  ```
  ./fail2ban
  ```
